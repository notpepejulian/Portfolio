AutoTest.es

Plataforma Educativa de Tests de Autoescuela

Plataforma web completa para realizar tests de autoescuela con generación automática de exámenes, gestión de preguntas DGT y sistema de autenticación.

Ver en GitHub Volver a Proyectos

Vista del Proyecto

Casos de Uso

Esta plataforma está diseñada para estudiantes de autoescuela que necesitan practicar y prepararse para el examen teórico. Algunos casos de uso incluyen:

Práctica de Exámenes

Realiza tests ilimitados con preguntas oficiales de la DGT para prepararte para el examen teórico.

Seguimiento de Progreso

Monitorea tu progreso y mejora tus resultados con estadísticas detalladas de tus exámenes.

Preparación Integral

Accede a un banco completo de preguntas oficiales con explicaciones y respuestas correctas.

Exámenes Cronometrados

Practica con exámenes que simulan las condiciones reales del examen oficial con tiempo limitado.

Perfiles de Usuario

Crea tu cuenta y guarda tu historial de exámenes, resultados y estadísticas personales.

Base de Datos DGT

Preguntas oficiales actualizadas de la Dirección General de Tráfico para máxima precisión.

Características Principales

Preguntas Oficiales DGT

Base de datos completa con preguntas oficiales de la Dirección General de Tráfico.

Generación Automática

Scripts Python para generar exámenes y tests automáticamente desde la base de datos.

Sistema de Autenticación

Gestión completa de usuarios con registro, login y sesiones seguras.

Estadísticas y Progreso

Seguimiento detallado del progreso con gráficos y métricas de rendimiento.

Docker Compose

Despliegue simplificado con contenedores Docker para desarrollo y producción.

Scripts de Gestión

Herramientas Python para insertar preguntas, generar tests y gestionar la base de datos.

Requisitos e Instalación

Requisitos

  • •Python 3.x para scripts y backend
  • •Astro para el frontend
  • •Docker y Docker Compose
  • •MySQL como base de datos

Cómo Ejecutar

1. Clonar el repositorio:

git clone https://github.com/notpepejulian/AutoTest.git

2. Iniciar con Docker:

docker-compose up

3. Ejecutar scripts de inicialización:

Usa los scripts Python para poblar la base de datos con preguntas DGT
