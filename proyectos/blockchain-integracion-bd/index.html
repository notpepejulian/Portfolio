MySQL-BigchainDB

Integración de MySQL y BigchainDB con API-REST y Tendermint

API REST diseñada para integrar una base de datos relacional MySQL con BigchainDB, proporcionando almacenamiento rápido y seguro combinado con inmutabilidad y trazabilidad.

Ver en GitHub Volver a Proyectos

Arquitectura del Proyecto

API REST

Node.js + Express

  • •Endpoints para crear y validar registros
  • •Gestión de conexiones MySQL y BigchainDB
  • •Cifrado y descifrado de datos sensibles

MySQL

Base de Datos Relacional

  • •Almacenamiento de datos estructurados
  • •Consultas rápidas y eficientes
  • •Relaciones entre tablas

BigchainDB

Base de Datos Distribuida

  • •Inmutabilidad garantizada
  • •Registro de transacciones y hashes
  • •Tendermint para tolerancia a fallos

Docker

Contenedores

  • •Gestión de MySQL, BigchainDB
  • •MongoDB (backend de BigchainDB)
  • •Tendermint (motor de consenso)

Flujo de Datos

1

Inserción de Datos

·Cliente realiza solicitud POST a la API REST

·Datos validados y almacenados en MySQL

·Se genera un hash y se registra en BigchainDB

·El ID de transacción queda actualizado en MySQL

2

Validación de Datos

Los datos en MySQL se comparan con el hash registrado en BigchainDB para garantizar integridad.

3

Cifrado y Descifrado

Los datos se cifran antes de almacenarse. Solo usuarios autorizados pueden descifrarlos. Registros inmutables en BigchainDB.

Endpoints Principales

MétodoEndpointDescripción
POST/create_transactionCrea un registro en MySQL y una transacción en BigchainDB. Devuelve el hash.
GET/validate/:idValida la integridad de un registro de MySQL comparándolo con BigchainDB.
GET/get_transaction/:transactionIdObtiene y descifra datos asociados a una transacción específica.

Casos de Uso

Este proyecto es ideal para aplicaciones donde la integridad y la autenticidad de los datos son fundamentales. Algunos casos de uso incluyen:

Cadena de Suministro

Trazabilidad completa e inmutable de productos desde el origen hasta el consumidor final.

Auditorías

Registro inmutable de transacciones y cambios para cumplimiento normativo y auditorías.

Registros Académicos

Certificación y verificación de títulos y credenciales académicas de forma segura.

Contratos Inteligentes

Registro de acuerdos y contratos con garantía de inmutabilidad y trazabilidad.

Historial Médico

Almacenamiento seguro e inmutable de registros médicos y tratamientos.

Certificaciones

Emisión y verificación de certificados digitales con garantía de autenticidad.
